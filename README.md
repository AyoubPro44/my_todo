## Overview  
This application allows users to create, view, edit, and delete tasks. It focuses on simplicity and productivity, offering a straightforward interface and data persistence so tasks remain saved across sessions.

## Features  
- Add new tasks with a title, optional description, and due date.  
- Mark tasks as completed or delete them when done.  
- View the list of tasks, filtered by status (e.g., active vs completed).  
- Optional: sort tasks by due date, priority, or tag.

- Create a new task using the “Add Task” form.
